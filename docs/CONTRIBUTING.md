# Как внести вклад в проект

## Для новых участников
1. Сделайте форк репозитория
2. Создайте ветку для задачи: `feature/название-задачи`
3. Выполните изменения и сделайте коммит
4. Отправьте Pull Request в ветку `main`

## Соглашения
- Коммиты на русском языке
- Одна задача — одна ветка
- Код должен быть читаемым и комментироваться при необходимости
